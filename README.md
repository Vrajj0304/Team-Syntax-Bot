# EcoSight
📝 Project title
Satellite Environmental Intelligence Platform for Smart Cities.

📝 Problem Statement
Despite the abundance of free environmental data from agencies like ISRO, NASA, and ESA, urban planners and civic bodies lack the integrated tools necessary to translate complex, fragmented satellite observations into actionable environmental policy.

Current barriers include technical data disharmony (varying resolutions and formats) and the absence of a systematic pipeline to monitor and mitigate critical urban challenges—such as heat islands, air pollution, and land degradation—in real-time.
